## 📋 PHASE 1 : SETUP (AVANT DE DÉPENSER)

### 1. Lier Google Ads ↔ Google Analytics 4

```
Google Ads → Outils et paramètres → Paramètres liés → Link GA4
```

**Pourquoi** : Permet de tracker au-delà des clics (pages vues, temps sur site, bounce rate, transactions réelles)

### 2. Configurer la conversion "Devis soumis"

**Étapes** :
1. Google Ads → Conversion → Créer une conversion
2. Source : "Clic sur site web"
3. Catégorie : "Lead"
4. Nom : "Devis soumis"
5. Valeur : 50€ (coût moyen d'acquisition pour toi)
6. Compter : "Chaque conversion"

**Résultat** : Google optimise ses enchères sur les **leads réels**, pas les clics aléatoires.

## 📈 DASHBOARD GOOGLE ADS À MONITORER

### QUOTIDIEN (5 min)

```
□ CTR (Click-Through Rate)
  Objectif : > 3% par campagne
  Si < 2% = mauvaise copie d'annonce ou landing page

□ CPC (Cost Per Click)
  Objectif : < 1.50€ moyen
  Si > 2€ = bid strategy pas optimisée

□ Impressions
  Objectif : > 1000/jour (sinon = budget trop bas)
```

### HEBDOMADAIRE (30 min)

```
□ ROAS (Return On Ad Spend)
  Objectif : > 4:1 (4€ revenue par 1€ dépensé)
  Calcul : (Conversions × Valeur) / Budget dépensé

□ Coût par conversion
  Objectif : < 40€ (peu importe combien tu factures)
  
□ Leads qualifiés
  Objectif : > 3/jour (13-19/mois)

□ Search Terms Report
  Action : Pause les mots-clés hors-sujet
```

### AJUSTEMENTS CHAQUE LUNDI

```
□ Identifie top 20% ad groups (meilleur CTR)
  → Double leur budget

□ Identifie bottom 20% ad groups (pire CTR)
  → Pause-les ou teste nouvelle copie

□ Review Search Terms
  → Ajoute « negative keywords » hors-sujet
  Exemple : "-emploi", "-gratuit", "-formation distance"

□ Vérifie Quality Score des mots-clés
  Si < 6 : mauvaise landing page ou copie peu pertinente
```
